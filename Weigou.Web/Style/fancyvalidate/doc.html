安装

在页面头部引用核心脚本： 

<script src="fancyValidate.min.js"></script>

(可选)如果需要额外的验证规则，则再引用附加脚本： 

<script src="fancyValidate.additional.min.js"></script>

还有包括错误消息的样式：

<link href="css/fancyValidate.css" rel="stylesheet"/>

然后在DOM加载完成时使用它：

<script>
$f.dom.ready(function() {
  $f("form1", {
    //各种选项...
  });
});
</script>

整体看起来就像这样：

<head>
  <link href="css/fancyValidate.css" rel="stylesheet"/>
  <script src="fancyValidate.min.js"></script>
  <script src="fancyValidate.additional.min.js"></script>
  <script>
  $f.dom.ready(function() {
    $f("form1", {
      //各种选项...
    });
  });
  </script>
</head>

选项配置

参数 类型 说明
rules Object 验证规则；如：
rules: {required: 1,digit: 1}
messages Object 验证规则对应的提示消息；如：
messages: {required: "请填写",digit: "必须是数字"}
(可选)ruleKey String 规则的key对应的元素属性；默认为：
ruleKey: "name"
(可选)ruleKeyAlter String 规则的key对应的元素属性(需要定义额外的不同于ruleKey时)；默认为：
ruleKeyAlter: null
(可选)getKey (element) Function 用于从元素获取规则的key；默认为：
getKey: null
(可选)ruleAttr String 规则内容对应的元素属性；默认为：
ruleAttr: "data-val"
(可选)validCls String 验证成功时标签的样式；默认为：
validCls: "success"
(可选)errorCls String 验证失败时标签的样式；默认为：
errorCls: "error"
(可选)errorElCls String 验证失败时元素的样式；默认为：
errorElCls: "fld-error"
(可选)validElCls String 验证成功时元素的样式；默认为：
validElCls: "fld-valid"
(可选)activeElCls String 元素获得焦点时的样式；默认为：
activeElCls: "fld-active"
(可选)errorElement String 显示提示信息的标签名称；默认为：
errorElement: "label"
(可选)appendLabel (label, element) Function 用于把提示信息标签添加到页面上；默认为：
appendLabel: $f.appendNext
(可选)findLabel (element, forId) Function 查找已有的提示信息标签；默认为：
findLabel: $f.findNext
(可选)labelText (label, message) Function 设置提示信息内容；默认为：
labelText: 内置的(设置innerHTML)
(可选)interceptInput Boolean 在输入时进行验证；默认为：
interceptInput: true
(可选)interceptSubmit Boolean 在表单提交时进行验证；默认为：
interceptSubmit: true
(可选)focusInvalid Boolean 验证失败时为元素获得焦点；默认为：
focusInvalid: true
(可选)autoTab Boolean 验证成功时为下一个元素获取焦点；默认为：
autoTab: false
(可选)submitRandom Boolean 发送随机数到服务器；默认为：
submitRandom: true
(可选)submitHandler (form, event) Function 自定义验证成功时的行为；默认为：
submitHandler: null
(可选)beforeSubmit (form, event) Function 表单提交前的行为；默认为：
beforeSubmit: null
(可选)debug Boolean 启用调试，用于输出一些信息；默认为：
debug: false
(可选)getList (element) Function 用于获取元素(主要是checkbox/radio)同组的元素列表；默认为：
getList: null
(可选)highlight (element) Function 验证失败时对元素进行操作，如设置样式；默认为：
highlight: 内置的
(可选)unhighlight (element) Function 验证成功时对元素进行操作，如设置样式；；默认为：
unhighlight: 内置的

内置的验证规则

名称 类型 说明
required Boolean/Number 必填；如：
required:1 required:true
compareTo String/HTMLElement 与其他表单元素的值相等；如：
compareTo:"pwd" compareTo:$f.dom.get("pwd")
pattern String/Regexp 正则表达式；如：
pattern:"^abc$" pattern:/^abc$/
fn String/Function 函数；函数名称、函数；如：
fn:"fn1" fn:function(value,element,param){...}
minlength Number 最小长度；如：
minlength:1
maxlength Number 最大长度；如：
maxlength:9
rangelength Array 长度范围；如：
rangelength:[1,9]
min Number 最小值；如：
min:1
max Number 最大值；如：
max:9
range Number 值范围；如：
range:[1,9]
numeric Boolean/Number 数字类型(包括小数)；如：
numeric:1 numeric:true
digit Boolean/Number 整数；如：
digit:1 digit:true
prefix String 前缀；如：
prefix:'u_'
suffix String 后缀；如：
suffix:'_d'
equal String/Number 相等；如：
equal:'abc' equal:123

appendLabel: $f.appendNext
$f.findLast 查找表单元素父节点的最后一个子节点，假如它是提示信息标签的话；如：
findLabel: $f.findLast
